


angular core concepts
--------------------------

1. data binding

    a. interpolation
        syntax: {{expression}}

    b. property/attribute binding
        syntax:
        <element [prop1]="expression"/>

    c. event binding
        syntax:
        <element (event-name)="eventHandler($event)"/>



2. directives

    => re-usable UI functionalities

    1. component  => <app-root>
    2. structural => ngIf,ngFor
    3. property/attribute  => ngClass
